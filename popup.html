FAQs

What about the "Bad response" error?

Each time you submit a keyword, a real search is performed on the selected Web Search Engine.
If too many searches are done in a short time, the Web Search Engine will block your actions for a while.
I suggest to diluite your searches over time. To help you on not loosing track, the latest keyword used is highlighted in the list.

I have tons of keywords to check, can this be automated?

No, for the same reason stated above.
